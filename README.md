# Comparative Genomics of SARS-CoV-2 vs SARS-CoV Tor2

## Data
- SARS-CoV-2 genome (MN908947.3)
- SARS-CoV Tor2 genome (AY274119.3)

## Alignment Method
- Full-genome alignment using MAFFT

## Results
- Alignment length: 30,016 bp
- Mutations detected: 6,319
- Divergence: 21.05%

## Scripts
- compare_sars_alignment.py calculates divergence positions and mutation count

## Files
- SARS_Aligned.fasta (Aligned genomes)
- compare_sars_alignment.py (Python script)
- Summary.pdf (Project overview and interpretation)
